# Áudio do Assistente de Voz

Coloque aqui o arquivo **welcome-bg.mp3** para a trilha de fundo da mensagem de boas-vindas.

- Nome do arquivo: `welcome-bg.mp3`
- Uso: trilha ambiente em loop, volume baixo (8–12 s antes da fala, depois ducking durante o TTS).
- Se o arquivo não existir, apenas a mensagem de voz (TTS) será reproduzida, sem música.
